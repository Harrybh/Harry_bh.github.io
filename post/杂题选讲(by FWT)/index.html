CF1254E Send Tree to Charlie

  • 有一棵包含$n$个节点的树，初始$i$号点上的标记是$i$
  • 随便钦定一个边的排列，然后依次对排列中的边，交换端点的标记
  • 给出你一部分点的最终的标记$a_i$，问有多少种确定剩下的点的最终标记的方式，使得存在一个边的排列能达到这种状态
  • $n\le 5\times 10^5$，答案对$10^9 + 7$取模

CF1254E Send Tree to Charlie

  • 提示：如果$\sum_{\text{i的最终标记给出}} dis(i,a_i) > 2n-2$必然无解

CF1254E Send Tree to Charlie

  • 每个点的所有邻边的相对操作顺序确定，就可以唯一确定最终的$a_i$
  • 我们考虑每一个给定了最终标记的$i$会造成什么限制：
    • 考虑$i\to a_i$这条路径，对于路径上的第一个点，路径上的第一条边是与这个点相邻的所有边中第一个被操作的；
    • 对于路径中间的点$u$，路径上与$u$相邻的两条边$x,y$，操作完$x$之后操作的下一条与$u$相邻的边必须是$y$；

  • 所以对每个点统计它的所有邻边的可能的排列顺序即可

CF1270H Number of Components

  • 有一个长度为$n$序列$a$
  • 有一张有$n$个点的无向图$G$，对于每一对$i< j$，$i,j$之间有边当且仅当$a_i < a_j$
  • 有$q$次操作，每次会修改序列中的某个位置的值
  • 问每次操作后，$G$的连通块数
  • $n,q\le 5\times 10^5$，保证任意时刻$1\le a_i \le 10^6$

CF1270H Number of Components

  • 观察发现，如果$i,j(i<j)$连通，那么对于任意的$k\in (i,j)$，$k$与$i,j$连通。
  • 证明：考虑连通块中从$i$到$j$的一条路径，若$k$在这条路径上结论显然成立；否则，这其中必定存在一条边$(u,v)$满足$u < k < v$，由于$a_u < a_v$，所以$a_u < a_k \vee a_k< a_v$一定成立
  • 所以，连通块一定是序列上的一段连续的区间

CF1270H Number of Components

  • 考虑计算：计算有多少个$p$，满足$[1,p]$和$[p+1,n]$之间没有边
  • 等价于$\forall x\in [1,p], y\in [p+1,n], a_x > a_y$
  • 考虑枚举$[p+1,n]$中的最大值$v$，记序列中小于等于$v$的值为$0$，大于$v$的值为$1$，则$p$合法的条件就是整个序列形如
  • 由于对于每个$v$而言，它对应的$01$序列是确定的，所以能够由它产生贡献的$p$只有一个
  • 所以可以直接统计有多少个$v$对应的$01$序列形如$111\cdots 111000\cdots 000$

CF1270H Number of Components

  • 为了方便处理，我们令$a_0 = + \infty, a_{n+1} = -\infty$
  • 对每一个$v$维护它对应的$01$序列中相邻的$10$对的数量，以及$v$是否作为序列中的某个元素出现；$v$合法的条件就是相邻的$10$对的数量为$1$
  • 由于$a_0 > v, a_{n+1} < v$，所以相邻的$10$对的数量$\ge 1$
  • 维护区间内$10$数量的最小值，以及取到最小值的位置的个数，即可得到合法的$v$的数量

AGC039F Min Product Sum

  • 一个$n\times m$的矩阵的权值定义如下：
    • 每个格子的权值定义为这个格子所在的行和列的所有元素的最小值
    • 整个矩阵的权值定义为所有格子的权值的乘积
  • 给出$n,m,K$，问所有的每个格子的取值都在$[1,K]$的$n\times m$的矩阵的权值和
  • 答案对给定的一个质数$D$取模
  • $1\le n,m,K\le 100, 10^8 \le D \le 10^9$

AGC039F Min Product Sum

  • 考虑权值的组合意义：一个矩阵$A$的权值等于，往另一个矩阵$B$填数使得$B_{i,j}$不大于$A$中与它在同行或者同列的任何一个数的方案数
  • 等价于：对于每一行，$B$的最大值小于等于$A$的最小值；对于每一列，$B$的最大值小于等于$A$的最小值
  • 等价于：枚举每一列$A$的最小值$Y_i$，并要求这一列的$B$的所有元素都不超过它；枚举每一行$B$的最大值$X_i$，并要求这一行的$A$的元素都不小于它

AGC039F Min Product Sum

  • 对于$A$中的每个格子，限制是：$A_{i,j}\ge X_i,A_{i,j} \ge Y_j$，且$j$列中至少有一个元素$A_{k,j}=Y_j$
  • 对于$B$中的每个格子，限制是：$B_{i,j}\le X_i,B_{i,j} \le Y_j$，且$i$行中至少有一个元素$B_{i,k}=X_i$
  • 对于某一个格子$A_{i,j}$，如果$X_i>Y_j$，就在枚举$X_i$的时候考虑它的贡献；如果$X_i<Y_j$，就在枚举$Y_j$的时候考虑它的贡献；如果$X_i=Y_j$，我们规定在枚举$Y_j$的时候考虑它的贡献（相当于是规定，对于$X_i=Y_j$，认为$X_i$更小）
  • 对于某一个格子$B_{i,j}$，如果$X_i\le Y_j$，就在枚举$X_i$的时候考虑它的贡献，否则就在枚举$Y_j$的时候考虑它的贡献。
  • 为了方便统计，我们先将行按照$X_i$排序，列按照$Y_i$排序

CF671E Organizing a Race

  • 有$n$个城市排成一排，第$i$个城市到第$i+1$个城市之间的道路长度是$w_i$
  • 每走一单位长度的路需要消耗一单位的油
  • 每个选手每一次经过城市$i$都会额外获得$g_i$单位的油；车的油箱是无限大的；特别地，选手出发前油箱是空的，也就是说如果一开始选手从$i$城市出发，则选手一开始会获得$g_i$的油
  • 一个区间$[l,r]$是好的，当且仅当：
    • 选手从$l$单向向右开到$r$，不会出现半路上没油的情况
    • 选手从$r$单向向左开到$l$，不会出现半路上没油的情况
  • 你可以让每个城市的$g_i$增加一些（或者不变），要求总增加量不超过$lim$
  • 问最长的好的区间的长度
  • $2\le n\le 10^5,0\le g_i, lim \le 10^9, 1\le w_i \le 10^9$

CF671E Organizing a Race

  • 答案没有单调性，不能二分
  • 如何判断$[l,r]$是否合法：从$l$开始往$r$走，当汽油不够的时候就给当前城市加汽油使得恰好够走到下一个城市；然后把剩下的汽油都加在$r$，再判断能否从$r$走到$l$。

CF671E Organizing a Race

  • 设$pre_i = pre_{i-1}+g_{i-1}-w_{i-1}$
  • $i$往右第一个无法到达的城市$x$，就是第一个满足$pre_x-pre_i<0$的$x$，我们记为$next_i = x$，若$x$不存在就记$next_i = n+1$
  • 由于我们在$x-1$加的油量让我们“恰好可以走到$x$”，所以从$i$到达$x$时油箱为空，所以下一个走不到的点就是$next_x$
  • 将$next_i$作为$i$的父亲，会得到一个树形结构，并且$\{n+1,n,n-1,n-2\cdots 1\}$是这棵树的一个dfs序

CF671E Organizing a Race

  • 对整棵树进行dfs，令当前走到的点为$l$
  • 对每个点维护$g_i’$表示如果从$l$出发要走到$i+1$需要在这个城市额外加的油量加上这个城市原来的油量，以及$suf’_i=suf’_{i-1}+g_i’-w_{i-1},cost_i = \sum_{l\le k \le i}g_k’-g_k$
  • 一个$r$合法当且仅当：
    • $(cost_r-(g’_r-g_r)) - \min_{l\le k<r} \{(suf’_r-(g’_r-g_r))-suf’_k\} \le lim$（减去$g’_r-g_r$是因为不需要到达$r+1$）
    • $cost_r\le lim$（后面那坨东西可能小于0）
  • 上面的表达式等价于$\max_{l\le k<r}\{suf’_k\} + cost_r - suf’_r\le lim\wedge cost_r\le lim$。

CF671E Organizing a Race

  • 从$next_i$走到$i$，会对$g’_k,suf’_k,cost_k$造成的影响是：
    • $g’_{next_i - 1}+=pre[i]-pre[next_i]$
    • $suf’_k += pre[i] - pre[next_i] (\forall k \ge next_i - 1)$
    • $cost_k += pre[i] - pre[next_i] (\forall k \ge next_i-1)$
  • 由于每一次对$suf’_r,cost_r$的修改量是相同的，所以$cost_r-suf’_r$对于每一个$r$都是一个定值，我们只需要维护对$suf’$的修改

CF671E Organizing a Race

  • 在线段树上对$r$二分，要支持快速查询一个区间内的$\min \{\max_{l\le k<r}\{suf’_k\} + cost_r - suf’_r\}$
  • 线段树维护单调栈，一次查询$O(\log n)$
  • 对于$cost_r\le lim$的限制，由于不合法的是一段后缀，对那段后缀的点的$suf’$都加上$\infty$然后直接做就可以了
  • 总时间复杂度$O(n\log^2 n)$。

  • 什么你说你不会线段树维护单调栈？好吧其实我也不会
  • 对每个区间，维护$r$在右儿子里面选，$l$为区间左端点的答案$ans[c]$

Comet OJ Contest#14 F 水平竖直线段

  • 平面上有$n$条水平线段和$n$条竖直线段，保证任意两条线段不在同一直线上
  • 考虑这样一张图$G$，$G$每个点都代表一条线段，两个点之间有边当且仅当这两条线段有交点
  • 对于每条线段，求出它是否是它所在连通块的割点
  • $2\le n\le 10^5$

Comet OJ Contest#14 F 水平竖直线段

  • 模拟tarjan的过程，也就是先求出一棵dfs树，再求出每个点的low
  • 求dfs树需要快速找到一个和$u$相邻且未被访问过的点
  • 求low则需要查询与一个点相邻的（包括它的父亲）所有点的深度最小值

Comet OJ Contest#14 F 水平竖直线段

  • 对水平和竖直线段分别构造函数式线段树，要求$i$条线段能够在第$[l_i,r_i]$棵线段树的下标为$i$的位置访问到
  • 令某个点的vis=1（实质上是删点）：可以直接在第$l_i$棵线段树中对应的叶子标记。访问到某个点的时候，如果它子树内的所有点的vis都是1，就把这个点的vis也标记为1。这一步是$O(n\log n)$的
  • low直接用函数式线段树做
  • 总时间复杂度$O(n\log n)$
